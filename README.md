areadinghelper
==============

an android app designed for a better reading experience

阅读助手
========

暂时没想到什么好名字啦。

这个应用设计的初衷是辅助阅读——通过设定计划提醒自己每天都要读书，并留下读书笔记。

目前实现的功能有添加图书，跟踪进度，留下书评。

目前是早期预览版，更多功能开发中！
